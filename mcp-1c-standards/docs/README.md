# Документация MCP 1C Standards

MCP-сервер со справочником стандартов разработки 1С (BSL). Работает офлайн, без выхода в сеть.

## Разделы

- **[Описание сервиса](service.md)** — назначение, возможности, инструменты (tools), откуда берётся контент.
- **[Добавление стандартов](adding-standards.md)** — пошаговая инструкция: как добавить новый стандарт в контент и в lookup.

## Быстрый старт

- Сборка: `go build -o mcp-1c-standards .`
- Подключение в Cursor: в MCP-настройках указать команду на путь к бинарнику `mcp-1c-standards`.
- Подробности — в [корневом README](../README.md).
